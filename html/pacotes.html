Pacotes Em Java

O que são pacotes em Java?

Java possui uma maneira simples e direta de organizar arquivos e permite usar várias classes com o mesmo nome no mesmo projeto. Essa forma de organização é chamada de embalagem. Java usa a palavra pacote para se referir a um pacote de uma classe.

A árvore hierárquica do pacote não possui regras restritas de limitação de quantidade de nós, apenas recomenda-se notação semelhante a da Internet, a fim de reduzir ao máximo a possibilidade de termos mais de um pacote de arquivos de mesmo nome.
